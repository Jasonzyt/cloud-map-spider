# cloud-map-spider

## Feature

- Auto fetch and download cloud maps from NSMC and etc...
- Can be easily extended to other meteorological service sites

## Usage

```shell
git clone https://github.com/Jasonzyt/cloud-map-spider.git
cd cloud-map-spider
pip install -r requirements.txt
python3 main.py
```
